# Y7 Worksheets — Interactive HTML Tools

## What This Is
- GitHub Pages site: yunshugao.github.io/y7-worksheets/
- Interactive HTML-based worksheet tools for Year 7 Technology (NSW)
- All 3 classes (7ECE, 7ECI, 7TECJ) use these materials

## Tech Stack
- Vanilla HTML/CSS/JavaScript — no frameworks, no build tools
- Each .html file is fully self-contained (styles + scripts inline)
- GitHub Pages deploys from main branch automatically
- Target devices: school Chromebooks + student phones (must be mobile-friendly)

## Contents
- T1-L01 through T1-L20: lesson worksheets for Engineering & Sustainable Design unit
- Write-It-Right.html: interactive writing/grammar correction tool
- design-process-interactive.html: interactive design process explorer
- index.html: landing page linking to all tools

## Rules
- Australian English always (colour, organise, behaviour, analyse)
- Student-facing: plain language (Year 7 reading level), encouraging tone
- Mobile-responsive: test at 375px width minimum
- Accessibility: sufficient contrast, keyboard navigable, screen-reader friendly
- No external dependencies — everything works offline once loaded
- Google Classroom embeddable — avoid features that require popups
